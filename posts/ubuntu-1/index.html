Ubuntu: 完善-1

2016年9月26日
| Ubuntu

Ubuntu 16.04 WPS无法输入中文

  1. 安装搜狗linux

    http://pinyin.sogou.com/linux/?r=pinyin

  2. 修改配置文件

    • 修改wps

      vim /usr/bin/wps
# 添加：
export XMODIFIERS="@im=fcitx"
export QT_IM_MODULE="fcitx"

    • 修改et

      vim /usr/bin/et
# 添加
export XMODIFIERS="@im=fcitx"
export QT_IM_MODULE="fcitx"

点击launcher图标最小化

gsettings set org.compiz.unityshell:/org/compiz/profiles/unity/plugins/unityshell/ launcher-minimize-window true

开启风扇

sudo find /sys | grep pwm
echo 255 | sudo tee /sys/devices/platform/asus-nb-wmi/hwmon/hwmon3/pwm1
